Hakkımızda

Güvenli Şifreleme Sistemi

Bu proje, güvenli şifreleme ve özet fonksiyonlarını kullanıcı dostu bir arayüzle sunmayı amaçlamaktadır.

Özellikler:

  • Eliptik Eğri Kriptografisi (ECC) ile metin şifreleme
  • SHA512 kullanarak metin ve dosya özeti oluşturma
  • Kullanıcı dostu arayüz
  • Tarayıcı tabanlı çalışma

Kullanılan Teknolojiler:

  • HTML5, CSS3
  • JavaScript
  • Web Crypto API
